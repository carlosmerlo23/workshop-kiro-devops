# Bienvenida y Objetivos del Workshop

## Duracion: 15 minutos (8:00 - 8:15)
## Fecha: Jueves 9 de julio de 2026

---

## Bienvenida

Bienvenidos al Workshop **DevOps con Kiro** disenado especialmente para el equipo de Carvajal.

En las proximas horas exploraremos como Kiro, el IDE potenciado por IA de AWS, puede transformar la forma en que implementamos practicas DevOps en nuestros proyectos.

---

## Que vamos a lograr hoy

### Al finalizar el Day 1, cada participante habra:

- [ ] Configurado Kiro como su herramienta DevOps asistida por IA
- [ ] Generado un pipeline de CI funcional usando prompts inteligentes
- [ ] Creado pruebas automatizadas con asistencia de Kiro
- [ ] Documentado su pipeline siguiendo buenas practicas
- [ ] Entendido como Kiro acelera el ciclo DevOps

### Si continuamos al Day 2:

- [ ] Generado plantillas de Infrastructure as Code (Terraform/CloudFormation)
- [ ] Implementado monitoreo basico y alertas
- [ ] Completado un Code Challenge DevOps en tiempo real

## Check rapido de prerequisites

Antes de empezar, verificar:

1. Kiro IDE instalado y abierto
2. Sesion iniciada en Kiro (cuenta AWS Builder ID)
3. Git CLI funcional (`git --version`)
4. Cuenta de GitHub accesible
5. Terminal integrada funcionando

### Verificacion rapida (que el facilitador muestra):

```bash
# En la terminal integrada de Kiro:
git --version
node --version   # o python --version
```
